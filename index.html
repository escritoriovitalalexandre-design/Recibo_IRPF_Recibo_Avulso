⚙️ Configurar banco de dados
Para usar o sistema online, informe as credenciais do seu projeto Supabase. Isso é feito uma única vez por computador.
1
Acesse supabase.com e faça login no seu projeto.
2
Vá em Settings → API. Copie a Project URL e a chave anon/public.
3
Se ainda não criou as tabelas, rode o arquivo criar_tabelas.sql no SQL Editor do Supabase.
